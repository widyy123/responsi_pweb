Proyek Digital Persona Finder ini sebagai sarana eksplorasi untuk mengasah materi dalam praktikum yang telah dilakukan, khususnya dalam penggunaan HTML, CSS, dan JavaScript.Saya memilih desain ini untuk memberikan kesan modern dan menonjolkan identitas saya sebagai mahasiswa Informatika. Selain fitur utama untuk mengecek persona, halaman ini juga berfungsi sebagai kartu profil digital yang merangkum tautan penting seperti GitHub, LinkedIn, dan Instagram. Dengan begitu, proyek ini tidak hanya menjadi bahan latihan teknis, tetapi juga berfungsi sebagai landing page ringkas untuk mengenalkan profil profesional saya.
